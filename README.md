# Turbine Builder

Solana programs and tools built during the Turbine cohort. Each week focuses on a different project, progressing from Rust fundamentals to full on-chain applications.

## Projects

| Week | Project | Description |
|---|---|---|
| 1 | [Rust Price Oracle](week-1/rust-price-oracle) | Terminal-based SOL price guessing game built in Rust |
| 3 | [Escrow](week-3/escrow) | Trustless SOL-USDC escrow program with atomic swaps |
| 3 | [Dungeon Vault](week-3/dungeon-vault) | Multiplayer vault game with MagicBlock Ephemeral Rollups and VRF randomness |
| 4 | [AMM](week-4/amm) | Constant-product automated market maker with PDA-secured vaults |
| 5 | [NFT Staking](week-5/nft-staking) | Stake MPL Core NFTs and earn SPL token rewards over time |
| 5 | [NFT Marketplace](week-5/nft-marketplace) | Full marketplace for MPL Core assets with SOL/SPL payments, offers, and rewards |

## Stack

- Rust / Anchor / Solana CLI
- Metaplex Core (MPL Core)
- TypeScript (tests)
- Axum / SQLx / Teloxide (week-7)
